# Aetherfall Realms

Original browser-based fantasy MMORPG foundation with server-authoritative multiplayer and hybrid offchain/onchain Berachain economy.

## Quick start
1. `npm install --workspaces`
2. `npm run dev:server`
3. `npm run dev:web`

See `docs/game-design.md` for architecture, economy, contracts, MVP scope, and roadmap.
